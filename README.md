# Personal Ledger

一个个人记账网页应用，支持手机、平板和电脑浏览器使用。当前默认使用本地数据模式；配置 Supabase 后可以切换到云同步。

## 本地运行

```bash
cd /Users/qi./Documents/personal-ledger-mvp
npm install
npm run dev
```

打开终端显示的地址，通常是：

```text
http://localhost:5173/
```

如果终端提示 `npm: command not found`，先安装 Node.js：

```bash
brew install node
```

## 日常使用

- 首页显示当前余额、本期收入、本期支出和最近记录。
- 记账页可以手动记收入或支出。
- 统计页可以按任意时间段查看总收入、总支出和分类汇总。
- 设置页可以填写初始本金，当前余额会按“本金 + 收入 - 支出”计算。

## 导入微信和支付宝账单

进入 `设置 -> 账单导入`：

1. 选择来源：微信或支付宝。
2. 选择导入方式：一般用 `追加记录`；如果想用新账单覆盖当前记录，选 `替换记录`。
3. 上传微信导出的 `.xlsx` 或支付宝导出的 `.csv` 文件。
4. 预览解析结果后点击 `确认导入`。

目前已适配微信支付账单流水文件和支付宝交易明细导出文件中带说明行、中文编码和“不计收支”记录的格式。
